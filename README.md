# OlympiaVault API

OlympiaVault API is a PHP web application for managing and exploring data about Slovak Olympians. The project includes a public catalog with filtering, sorting and pagination, detailed athlete pages, user authentication, Google OAuth login, two-factor authentication, and login history.

The application also provides a REST API protected by JWT authentication. Authenticated users can create, update, delete and import Olympian records through a private CRUD interface that communicates directly with the API.

## Features

- Public list of Slovak Olympians
- Filtering, sorting and pagination
- Athlete detail pages
- Local login and registration
- Google OAuth2 authentication
- Two-factor authentication
- Login history
- XLSX import
- REST API for Olympian records
- JWT-protected API endpoints
- Private CRUD dashboard
- JSON bulk import
- API documentation page
